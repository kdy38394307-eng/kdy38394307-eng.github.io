# 음악 추천 사이트

## 폴더 구성

```
src/
├── components/   # 화면 컴포넌트
│   └── App.jsx
├── services/     # 외부 서비스 연결
│   └── firebase.js
├── styles/       # 화면 스타일
│   ├── App.css
│   └── global.css
└── main.jsx      # 앱 시작 파일
```

## 실행

```bash
npm install
npm run dev
```

## 주요 기능

- 오늘의 랜덤 음악 추천 및 좋아요
- 하루 한 번 노래와 사연 추천
- 같은 곡 아래에 모이는 청취자 사연
- 사연이 많아질 때 자동 스크롤
